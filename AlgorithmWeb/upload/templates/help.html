About our platform

In order to facilitate the clustering analysis of scRNA-seq data, we developed a user-friendly platform and the main contents are as below :

  • 15 scRNA-seq datasets
  • 15 scRNA-seq data clustering algorithms
  • Our newly developed approach IRWG
  • Python package with 16 scRNA-seq data clustering algorithms

Data Sources

Here 20 scRNA-seq datasets are provided on our website system. These datasets have been processed into the same format. In detail, there are two files for each dataset, one is gene expression file and another is cell label file. The gene expression file can be seen as a numerical matrix whose rows are genes and columns are cells. The cell label file contains two columns, the first column is cell name which is consist with the column names of gene expression file, and the second column is label of cell. All of these datasets can be downloaded from our website for further analysis.

Existing clustering methods

There are 15 existing scRNA-seq data clustering methods integrated on our website system.

CategoryMethods
Distance-based RaceID, pcaReduce, cellTree, CIDR, SINCERA, SC3
Density-based GiniClust
Graph-based SNN-Cliq, Seurat
Matrix-based NMF, BackSPIN
Probability model-based BISCUIT, DIMM-SC
Deep learning-based scVI, DCA

Users can get the corresponding clustering results by uploading their own data to the website.

You can get the results as following steps:
(1) Upload your scRNA-seq data which is a numeric matrix of expression values, where rows are genes, and columns are cells as a csv file.


(2) Fill in the parameter K which means the number of clusters.

(3) Select the clustering methods (one or more).

Once you finish steps above, you can get clustering results as a zip file at current web page or we will send it to your email soon.

New method IRGW

The new method IRWG (Integrated clustering with Random Walk on Graph) can also be applied. An example of GSE52583 dataset is given as this. The running steps are the same as other methods, in addition, three graphs are shown for users intuitively :
(1) Heat map
This figure is generated from cell similarity matrix, in which the element equals to the number of methods which classify corresponding cells into the same clusters.
(2) ARI bar
This figure is generated from clustering result with 5 methods (SC3, CIDR, pcaReduce, SINCERA and IRGW). According to comparing the predicting results with true labels of data, ARI (Adjusted Rand Index) can be calculated.
(3) Two-dimensional presentation of data
This figure is generated from the scRNA-seq data. ZIFA(Zero Inflated Factor Analysis) method is applied to mappings the data in multidimensional space to a two-dimensional and visualization of data, where different colors represent different cell types.
If you upload your data satisfies the required format, you can see these three figures for your own data soon.
